# LdeLuipy

Sitio estático del hub personal: subdominios, proyectos, documentación y enlaces.

**Sitio en producción:** [https://ldeluipy.es/](https://ldeluipy.es/)

## Contenido del repositorio

- `index.html` — página principal del hub
- `uses.html` — página de usos/herramientas
- `style.css`, `sw.js` — estilos y service worker (PWA ligera)
- `data/hub.json` — datos del hub (enlaces y secciones)
- `manifest.webmanifest`, `favicon.svg` — manifest e icono

Despliegue habitual: servir la raíz del proyecto como documento web estático.
